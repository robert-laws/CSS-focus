Web Fonts

@font-face Rule

Web fonts adds great flexibility to developers to use more than just system fonts when developing a design for their website. Used with the CSS3 @font-face rule, it's possible to add local or web-based fonts for use on a webpage.

Example Text

Web-Fonts in Action

Subtitle Information - no font style added

Lorem ipsum dolor sit amet consectetur adipisicing elit. Nulla, repudiandae.

Tempora nesciunt. Enim, est quod velit vel ratione iste provident? Vel!

Example HTML Code

HTML
<h3>Web-Fonts in Action</h3>
<h4>Subtitle Information - no font style added</h4>
<p>Lorem ipsum dolor sit amet consectetur adipisicing elit. Nulla, repudiandae.</p>
<p>Tempora nesciunt. Enim, est quod velit vel ratione iste provident? Vel!</p>

@font-face style declaration in stylesheet

CSS
@font-face {
  font-family: ChunkFive;
  src: url('../fonts/Chunkfive-webfont.woff') format('woff');
}
CSS
@font-face {
  font-family: RalewayThin;
  src: url('../fonts/raleway_thin-webfont.woff') format('woff');
}

Application of @font-face

CSS
h3.webfont {
  font-family: ChunkFive, sans-serif
}
CSS
p {
  font-family: RalewayThin, sans-serif;
}
