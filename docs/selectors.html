Using Selectors

Attribute Selectors

CSS3 builds on the selectors available from CSS1 and CSS2. In CSS3 there are selectors that target class, type, and attributes when making selections. CSS3 adds 3 additional attribute selectors. These build on the attribute selectors from CSS level 2. They are the beginning, ending, and arbitrary attribute selectors.

Here are some images will be used to demonstrate how attribute selectors work

Example Images

Example HTML Code

HTML
<img src='images/square.png' alt='square' title='image of a square'>
<img src='images/triangle.png' alt='triangle' title='image of a triangle'>
<img src='images/circle.png' alt='circle' title='image of a circle'>
<img src='images/line.png' alt='line' title='image of a line'>

Simple Attribute Selector

CSS
[alt] {
  border: 1px dashed #000;
}

Exact Attribute Value Selector

CSS
[alt='triangle'] {
  border: 2px solid #F00;
}

Partial Attribute Value Selector

CSS
[title~='blue'] {
  border: 3px dotted #F00;
}

Beginning Substring Atrribute Value Selector

CSS
[title^='orange'] {
  border: 4px solid darkorange;
}

Ending Substring Atrribute Value Selector

CSS
[src$='png'] {
  border-top: 5px solid green;
}

Arbitrary Atrribute Value Selector

CSS
[title*='image'] {
  border-bottom: 10px double yellowgreen;
}

General Sibling Combinator

Building on the concept of the adjacent sibling combinator, the general sibling combinator is new in CSS3.

Example Text

Heading Number 4

Lorem ipsum dolor sit amet consectetur adipisicing elit.

Pariatur voluptatibus aperiam dolorum debitis similique ipsum quasi, pariatur voluptatibus.

Example HTML Code

HTML
<article>
  <h4 class='comb_heading'>Heading Number 4</h4>
  <p class='comb_text_one'>Lorem ipsum dolor sit amet consectetur adipisicing elit.</p>
  <p class='comb_text_two'>Pariatur voluptatibus aperiam dolorum debitis similique ipsum quasi, pariatur voluptatibus.</p>
</article>

Adjacent Sibling Combinator

CSS
h4 + p {
  font-style: italic;
}

General Sibling Combinator

CSS
h4 ~ p {
  color: blue;
}
