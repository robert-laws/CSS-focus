Text Effects and Typographic Styles

Applying Dimensional Effects: text-shadow

The text-shadow property will apply a shadow effect to text, giving the user a sense of 3 dimensionality.

Text-Shadow Property

CSS
element {
  text-shadow: x y blur-radius color
}

Most basic application of the text-shadow property

CSS
p {
  text-shadow: -3px -3px;
}

Addition of color value

CSS
p {
  text-shadow: -3px -3px #CCC;
}

Addition of blur-radius value

CSS
p {
  text-shadow: -3px -3px 3px #CCC;
}

Use word-wrap to break if the text would extend beyond the containing element. The accepted values are normal or break-word

CSS
p {
  word-wrap: break-word;
}
