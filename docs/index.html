CSS3

Introduction

This website is an exploration of the features and implementation of CSS3. The website aims to describe in words, code, and implementation many of the most prominent and useful features of CSS3. Much of the content of this website is inspired by the information from The Book of CSS3, 2nd Edition by Peter Gasston. The book is available for purchase at amazon.com.

Code Examples

Conventions used to make examples and descriptions on this website include inline code snippets, such asbody { background-color: #F00; }to display code in the flow of the text.

For more substantial code, extending over multiple lines, the preformatted elementpreis used to display the code as a separate block to make it easier to read. For example:

CSS
body {
  background-color: red;
}
CSS
body {
  margin: 10px
  padding: 0
  background-color: red;
  border: 1px solid #000;
}

All the code including HTML, CSS, and Javascript for this website is coded from scratch without the use of any frameworks. This allow focus on the features of CSS3 without external code getting in the way.
